mongodb的使用

一.安装

快速讲解(windows安装)

1.下载mongodb安装包windows版，安装（可以自定义选择安装路径）,bin文件夹介绍：

2.在根目录下创建data文件夹，里面创建一个db文件夹用于存放数据。一个log文件夹用于存放日志

3.存放路径：mongod –dbpath f:\mongo\data\db

4.根目录创建mongod.congfig配置文件夹,输入：dbpath=f:\mongo\data\db
logpath=f:\mongo\data\log\mongod.log

5.配置：管理员启动cmd，输入mongod –config f:\mongo\mongod.config –install –serviceName “MongoDB”

6.启动：net start MongoDB

linux安装

ls进入，tar zxvf解压，mv移动,–dbpath,–logpath,–fork:后台执行

二.基本命令

1.show dbs:查看数据库

2.use dbname:使用某个数据库

3.show tables：展示collections

4.use dbname:隐式创建库

5.db.createCollection(“name”):创建collections

6.db.collection.insert({key:value}):直接创建collection

7.db.tablename.find():查看所有数据

8.db.collectionname.drop():删除

9.db.dropDatabase():删除创建的库

三.增、删、改、查

1.增

db.collectionname.insert({});

db.collectionname.insert([{},{}])

2.删

db.collectionname.remove({})
查询表达式是个json对象，删除匹配的数据，不传json，全部删除

db.collectionname.remove({json},true)
只删除对应所有数据中的一项

删除库

use name->db.dropDatabase()

3.改

db.collectionname.uodate({查询json},{改正后的值})
会全部改变

db.collectionname.update({查询json},{$set:{更改部分}})$set：只更改部分内容

$unset:删除某列

$rename:重命名

$inc:增长某列

multi:true/false:修改多行
db.collectionname.update({json查询},{新增},{multi:true})

upsert:true:如果有则修改，如果没有则添加
db.collectionname.update({json查询},{新增},{upsert:true})

$setOnInsert:true:当upsert为true,并且发生insert操作时,可以用来补充字段

db.collectionname.update({json查询语句},{$set:{},$setOnInsert:{}},{upsert:true})

4.查

db.collectionname.find()

db.collectuionname.find({json查询语句})

prop:1:查询某个属性prop

db.collectionname.find({},{prop:1})

prop:1,prop2:0:查询prop1,不查询prop2

db.collectionname.find({},{prop1:1,prop:0})

db.collectionname.find({查询语句},{name:1}):查询某种条件下得name

四.查询表达式

1.db.collectionname,find({id:{$ne:2}},{age:1})

查询id！== 2的所有数据的age属性

$gt:大于多少

$lt:小于

$lte:小于等于

$gte:大于等于

$in:in

{$in:[]}

$nin:not in

$or:or

$and:and

$not:not

$nor:全部不满足

$mod:求余为真

$mod:[5,0]:除5于0

$exists:含有age属性:{$exists:123}

$type:查询不同格式

$all:所有单元都要匹配

$where:{$where:’this.price> 0 && this.price < 100’}:不推荐，将数据转换成json对象

$regex:{price:{$regex:/^123/}}不推荐

10<=price<=200:
db.goods.find({$and:[{shop_price:{$lte:200}},{shop_price:{$gte:10}}]},{shop_price:1})

不属于3并且不属于11:

db.goods.find({$and:[{shop_price:{$ne:3}},{shop_price:{$ne:11}}]},{shop_price:1})

db.goods.find({shop_price:{$nin:[3,11]}})

db.goods.find({$nor:[{shop_price:3},{shop_price:11}]})

五.游标

查询的接口。可以逐条读取

游标操作：

var str = db.goods.find()

str.next()

str.hasNext()

str.forEach(callback())

分页中的使用方法:
跳过9992行

db.goods.find().skip(9992);

跳过9992取10条

db.goods.find().skip(9992).limit(10);

toArray(4):取出数据的第四行，减少使用，内存

六.索引

提高查询速度，降低写入速度

db.collectionname.ensureIndex({age:1/-1,name:1/-1}):创建索引,升序/降序

db.collectionname.getIndexes():查看索引

db.colectionname.dropIndex({age:-1})/db.collectionname.dropIndexes():删除索引

db.collectionname.ensureIndex({parent.children:1}):子文档索引

索引性质

unique:唯一索引

db.collectionname.ensureIndex({name:1},{unique:true})

sparse:稀疏索引，排除没有的列，只给存在的添加索引

hash索引:db.collectionname.ensureIndex({name:’hashed’}),范围查询更高效

db.collectionname:reIndex():重建索引,减少碎片

七.用户管理

1.创建用户

db.createUser({user:name,pwd:’password’,roles:[‘readWrite’]});

db.auth(“name”,”passwoed”):认证
